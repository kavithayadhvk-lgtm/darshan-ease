# DarshanEase

DarshanEase is a Temple Darshan Booking System built using microservices architecture.

## Services

- Frontend: React.js
- API Gateway: Node.js + Express.js
- Auth Service: JWT + bcrypt.js
- Temple Service
- Darshan Service
- Slot Service
- Booking Service
- Ticket Service
- Database: MySQL

## Ports

| Component | Port |
|---|---:|
| Frontend | 3000 |
| API Gateway | 5000 |
| Auth Service | 5001 |
| Temple Service | 5002 |
| Darshan Service | 5003 |
| Slot Service | 5004 |
| Booking Service | 5005 |
| Ticket Service | 5006 |
| MySQL | 3306 |

## Run

Install dependencies inside each service folder:

```bash
npm install
npm start
```
